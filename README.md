# Base Repository for Reporting R# Inspect Code Bugs

To reproduce bug, call the following cmd files from the VS Developer Command prompt:

`$ 1download-inspectcode.cmd`

`$ 2build.cmd`

`$ 3inspectcode.cmd`

Output will be written to `resharper.xml`.
